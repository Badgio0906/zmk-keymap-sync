GitHub 接続設定

PAT（Personal Access Token）とは？
GitHub API を使ってキーマップファイルを読み書きするための認証トークンです。
パスワードの代わりに使用します。

取得方法：
1. GitHub にログイン
2. 右上のアイコン → Settings → Developer settings
3. Personal access tokens → Tokens (classic) → Generate new token
4. 権限: repo にチェック → Generate token
5. 表示されたトークン（ghp_xxx...）をコピーして下に貼り付け

必要な権限: repo（読み書き） → GitHub でトークンを発行する（ブラウザで開く）
